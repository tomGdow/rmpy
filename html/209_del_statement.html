del statement
The Python del keyword is used to delete objects.
Its syntax is:
    del obj_name
Here, obj_name can be variables, user-defined objects, lists, items within lists, dictionaries etc.
Deletion is recursively defined very similar to the way assignment is defined.
Rather than spelling it out in full details, here are some hints.
Deletion of a target list recursively deletes each target, from left to right.
Deletion of a name removes the binding of that name from the local or global namespace, depending on whether the name occurs in a global statement in the same code block.
If the name is unbound, a NameError exception will be raised.

    a = 4
    del a
    del a
    //    NameError: name 'a' is not defined

Examples
Example 1: Class 
    class MyClass:
        a = 1
        def func(self):
           print('hello')

    inst = MyClass()
    inst.a          //    1
    inst.func()     //    hello
    print(MyClass)  //    <class '__main__.MyClass'>
    del MyClass
    del MyClass
    //    Traceback (most recent call last):
    //      File "<stdin>", line 1, in <module>
    //    NameError: name 'MyClass' is not defined
Example 2: delete list items 

    my_list_two              //     [10, 20, 40, 50, 60, 70, 80, 90]
    del my_list_two[2]
    my_list_two              //     [10, 20, 50, 60, 70, 80, 90]
    del my_list_two[0:3]
    print(my_list_two)       //     [60, 70, 80, 90]
    del my_list_two[:]
    my_list_two              //     []
Example 3: delete tuple 

    my_tuple=(1,2,3)
    del my_tuple

    my_tuple=(10,20,30)
    del my_tuple[0]
    //    Traceback (most recent call last):
    //      File "<stdin>", line 1, in <module>
    //    TypeError: 'tuple' object doesn't support item deletion
Example 3: delete from dictionary 

    mydict = dict(name='john', profession='programmer', age='25')
    mydict    //     {'name': 'john', 'profession': 'programmer', 'age': '25'}

    del mydict['profession']
    mydict    //     {'name': 'john', 'age': '25'}
