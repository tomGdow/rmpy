Armstrong number
An Armstrong number is a positive integer of n digits such that the sum of the nth powers of its digits is the number itself.
That is,
$$ abcd... = a^n + b^n + c^n + d^n ... $$
$$ 153^3 = 1^3 + 5^3 + 3^3 $$
So a 3-digit number will be an Armstrong number if the sum of the cubes of its digits is equal to the number itself.

    def intoDigits(int):
      digits =[]
      while int > 0:
        digit = int % 10
        digits.insert(0,digit)
        int //= 10

      return digits

    myin=input("Enter a digit")   // 123456789
