unpacking: * and ** operators
xx
Unpacking with * and ** operators
two operators * (for tuples) and ** (for dictionaries)
The * operator is an unpacking operator that will unpack the values from any iterable object
lists, tuples, string are examples
Syntax 
          Syntax One
Examples
Unpacking (function) 

    def printme(a,b,c):
        print(f" I am {a},{b},{c}")
    printme(*[1,2,3]) // I am 1,2,3 // Unpack in fn call // unpack args
Packing (function) 
    def sum_me(*args):
        return sum(args)  // pack into TUPLE // pack params
    sum_me(2,3,3,5) // 13
Unpacking (String) 
    first, *middle, last ='thomas'
    first    //  't'
    middle   //  ['h', 'o', 'm', 'a']
    last     //  's'
Unpacking (list) 
    f,*m,l=1,2,3,4,5

    f // 1
    m // [2, 3, 4]
    l // 5
Unpacking (Dictionary) 
    dict = {'a':1, 'b':2, 'c':3}

    def fn(a,b,c):
        print(a,b,c)

    fn(**dict) // 1 2 3
    BUT ...

    dict2 = {'x':10, 'y':20, 'z':30}
    fn(**dict2)
    ------------------------------------------------------
    TypeError: fn() got an unexpected keyword argument 'x'

    // argument names MUST correspond
Packing (Dictionary) 
    dict = {'a':1, 'b':2, 'c':3}

    Example 1

    def fn3(**args):
      for x in args.values():
        print(x, end=',')
    fn3(a=1, b=2, c=3) // 1,2,3,

    Example 2
    def fn4(**args):
      for x in args:
        print(x, end=',')
    fn4(a=1, b=2, c=3) // a,b,c,
Packing (Dictionary) 
    dict = {'a':1, 'b':2, 'c':3}

    Example 1
      def fn3 (**arg):
          for x in arg.values():
      print(x, end=' ')

      fn3(**dict)  // 1 2 3

    Example 2
      def fn4 (**arg):
          for x in arg:
      print(x, end=' ')

      fn4(**dict) // a b c
Packing into list (Important!) 
             
            *names, = "Tom", "Dick", "Harry"
            names // ['Tom', 'Dick', 'Harry']
            type(names)  // list

            // trailing comma after *names: left side must be a tuple or list
