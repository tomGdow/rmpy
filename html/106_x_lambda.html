lambda
Small anonymous functions can be created with the lambda keyword.
Lambda functions can be used wherever function objects are required.
They are syntactically restricted to a single expression.
Semantically, they are just syntactic sugar for a normal function definition.
Like nested function definitions, lambda functions can reference variables from the containing scope:
Janani Ravi (Skillsoft) on Lambda
✓ Although lambda functions are anonymous, you can assign a name to them just like other functions

    myfn = lambda x: x * x
✓ Like other functions, lambda functions are first-class citizens in python
× Cannot have if or other conditional statements in a lamdba
× This includes an assert statement!
✓ lambdas: define, invoke and discard! This is the idea

    (lambda x : x * 2)(10)
× No keywords (such as assert) within the body of lambda function

    def check_if_even(arg):       
    assert arg % 2 == 0

    check_if_even(2)              //   Does Nothing
    check_if_even(3)              //   !! Assertion Error
    lambda x : assert x % 2 == 0  //   !! SyntaxError: invalid syntax
✓ Can use keyword arguments with lambda functions
    (lambda x : x * x)(x=10)    //    100
✓ Can use default argument values

    (lambda chem, phys, biol=40 : chem + phys + biol)(10,20)            //  70
    (lambda chem, phys, biol=40 : chem + phys + biol)(chem=22,phys=20)  //  82
✓ Can have variable-length input arguments

    (lambda *args : sum(args))(1,2,3,4,5)    //     15
Syntax 

          lambda variable, ... : expression

          variable, …       //   Optional
          expression        //   Required

          OR

          lambda arguments : expression

          OR (Janani Ravi, Skillsoft) 

          lambda <input arguments> : <expression>
Examples
Example 1 (map) 
    f = lambda x: x*x
    lst = range(4)
    sqlst = map(f, lst)
    list(sqlst) // [0, 1, 4, 9]
Example 2 (filter) 
    l=range(1,10)
    list(l)      //  [1, 2, 3, 4, 5, 6, 7, 8, 9]
    ml = map(lambda x: x%2 == 0, l)
    list(ml)    // [False, True, False, True, False, True, False, True, False]
Example 3 (String memory address) 
    str="thomas"
    print(lambda str :str)
    <function <lambda> at 0x000001E62A88BDC0>
Example 4 (lambda within function definition) 
    def mult(n):
      return lambda x : x*n

    mult5 = mult(5)
    mult5(100)    //   500
    mult5
    <function mult.<locals>.<lambda> at 0x000001E62A8A5670>
Example 5 (sort strings with key function) 

    ids=['id1', 'id2', 'id30', 'id3', 'id22', 'id100']                      

    print(sorted(ids))  // ['id1', 'id100', 'id2', 'id22', 'id3', 'id30']

    list(map(lambda x: int(x[2:]), ids)) //  [1, 2, 30, 3, 22, 100]
    [
    print(sorted(ids, key=lambda x : int(x[2:]))) 
                                 // ['id1', 'id2', 'id3', 'id22', 'id30', 'id100']
Example 6 (List Comprehensions as Alternative to Lambda) 

    list(map(lambda x : x.capitalize(), ['dog', 'cat', 'mouse']))
    // ['Dog', 'Cat', 'Mouse']

    [x.capitalize() for x in ['dog', 'cat', 'mouse']]
    // ['Dog', 'Cat', 'Mouse']

    list(filter(lambda x: x%2==0, range(10)))
    // [0, 2, 4, 6, 8]
    [x for x in range(10) if x%2==0]
    // [0, 2, 4, 6, 8]
Notes
Historical Note
Functional languages get their origin in mathematical logic and lambda calculus
Imperative programming languages embrace the state-based model of computation invented by Alan Turing
The two models of computation, lambda calculus and Turing machines, can be translated into each another.
This equivalence is known as the Church-Turing hypothesis.

key functions and lambda
Key functions in Python are higher-order functions that take a parameter key as a named argument.
key receives a function that can be a lambda.
This function directly influences the algorithm driven by the key function itself.
