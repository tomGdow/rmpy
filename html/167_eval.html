eval
The expression argument is parsed and evaluated as a Python expression using the globals and locals dictionaries as global and local namespace
The arguments are a string. and optional globals and locals
If provided, globals must be a dictionary.
If provided, locals can be any mapping object.
If both dictionaries are omitted, the expression is executed with the globals and locals in the environment where eval() is called
eval can also be used to execute arbitrary code objects (such as those created by compile()).
In this case pass a code object instead of a string.
WARNING: If you allow users to input a value using eval(input()), the user may issue commands to change file or even delete all the files using the command: os.system('rm -rf *')
Syntax 
    eval(expression[, globals[, locals]])
Examples
Example 1 
            eval ('tom +2', {},{'tom':3}) // 5
Example 2 
    eval ('x+4', {}, {'x':200})        //   204
    eval ('x+4', {'x':44}, {'x':200})  //   204   // Locals take precedence
    eval ('x+4', {'x':44})             //   48
Example 3 
    //  Rafiq Wayani (Skillsoft)

    x = 4
    eval('x+100')    //   104
    x                //   4
    eval('x==2')     //   False
