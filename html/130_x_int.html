int([x], base=10)
Return an integer object constructed from a number or string x
Return 0 if no arguments are given.
x.__trunc__()
For floating point numbers, this truncates towards zero.
 
      int(10.2)        //     10
      int("10.3")      //     ValueError: invalid literal
      int("10")        //     10
      int('010', 8)    //     8
      int('1000',2)    //     8
      int(1000,2)      //     TypeError   !! Note
      int(-10.9)       //    -10
