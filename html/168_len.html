len
Python len() is a built-in function that returns the length (number of items) of an object.
len() function takes only one parameter as argument
The parameter can be a sequence, such as:
  • string
  • bytes
  • tuple
  • list
  • range
or a collection, such as:
  • dictionary
  • set
  • frozen set
__len__() method
The difference between __len__() and len() method:
len(s) is a built-in Python method which returns the length of an object.
__len__() is a special method that is internally called by len(s) method to return the length of an object.
The Python len() function can be interpreted as:
 
          def len(s):
              return s.__len__()
Syntax 

          len(s)

            //  s (required) – object whose length is to be returned
Examples
Example 1 
    ages =[0, 11, 43,12, 10]
    i=0
    for  i in range(0, len(ages)):
      print(ages[i], end=' ')
      
    //    0 11 43 12 10
