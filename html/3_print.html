print
 
           print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
Print objects to the text stream file, separated by sep and followed by end.
sep, end, file and flush, if present, must be given as keyword arguments.
All non-keyword arguments are converted to strings like str() does and written to the stream, separated by sep and followed by end
Both sep and end must be strings; they can also be None, which means to use the default values.
If no objects are given, print() will just write end.
Whether output is buffered is usually determined by file, but if the flush keyword argument is true, the stream is forcibly flushed.
Multiple values can be displayed as comma-separated list, separated by single space by default
alternative separator: by adding sep parameter, eg sep='**'
invisible \n at end of line //default

Examples
Example 1 (General) 

  price  = 3.95
  widgets = 5

  (1) print('The price of the widget is', price, 'and widgets is', widgets, end='!!')
      //    The price of the widget is 3.95 and widgets is 5!!

  (2) print('The price of the widget is ' +str(price) + ' and widgets is ' + str(widgets), end='!!')
      //    The price of the widget is 3.95 and widgets is 5!!

  (3) print(f'The price of the widget is {price} and {widgets}')
      //    The price of the widget is 3.95 and 5

  (4) print("The price of the widget is {} and widgets is {}".format(price, widgets), end='!!')
      //    The price of the widget is 3.95 and widgets is 5!!
