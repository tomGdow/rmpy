map
The definition of map is: 
    map(function, iterable).
It applies a function to every item in the iterable.
Syntax 
          Syntax One
Examples
Example 1 
    lst = [1,2,3,4]
    f = lambda x : x *x
    list(map(f, lst)) //  [1, 4, 9, 16]
Notes
Deprecation note

map is deprecated (Guido van Rossum reference)
