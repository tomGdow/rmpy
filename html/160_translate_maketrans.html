string.translate(table)
The string translate() method returns a string where each character is mapped to its corresponding character in a translation table.
The translate() method uses a translation table to replace/translate characters in a given string
The translation table is created by the maketrans()method
translate() takes a single parameter
string.maketrans(x, y, z)
The maketrans() method returns a mapping table that can be used with the translate() method to replace specified characters
param x
Required
If only one parameter is specified, this has to be a dictionary describing how to perform the replace.
If two or more parameters are specified, this parameter has to be a string specifying the characters you want to replace.
param y
Optional
A string with the same length as parameter x
Each character in the first parameter will be replaced with the corresponding character in this string.
param z
Optional
A string describing which characters to remove from the original string
Python Docs: static str.maketrans(x[, y[, z]])
This static method returns a translation table usable for str.translate()
If there is only one argument, it must be a dictionary mapping Unicode ordinals (integers) or characters (strings of length 1) to Unicode ordinals, strings (of arbitrary lengths) or None.
Character keys will then be converted to ordinals.
If there are two arguments, they must be strings of equal length, and in the resulting dictionary, each character in x will be mapped to the character at the same position in y
If there is a third argument, it must be a string, whose characters will be mapped to None in the result.
Syntax 
string.translate(table)
Examples
Example 1 
            "thomas".translate({97: 65})     // ThomAs
Example 2 
    "Thomas".maketrans('sa', 'SA')  //  {115: 83, 97: 65}
Example 3 
    mystr="Thomas"
    table=mystr.maketrans('a', 'A', 'h')
    mystr.translate(table)    //   'TomAs'
