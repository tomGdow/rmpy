Floor Division
// is called the floor division operator
$$ n // d = floor(\frac{n}{d}) $$
$$ 10 // 3 = 3 $$
$$ 10 // (-3) = -4 $$
$$ (-10) // 3 = -4 $$
note that sign on divisor/dividend does not apply
x // y
Floored quotient of x and y
Also referred to as integer division.
The resultant value is a whole integer, though the result’s type is not necessarily int.
The result is always rounded towards minus infinity
 
    1//2 is 0
    (-1)//2 is -1
    1//(-2) is -1
    (-1)//(-2) is 0

Examples
Example 1 
      >>> 10//3
      3
      >>> 10//(-3)
      -4
      >>> 10//-3
      -4
      >>> -10//3
      -4
      >>> (-10)//3
      -4
Example 2 
    >>> 2//3
    0
    >>> 2.0//3
    0.0
    >>> 1.3//1.0
    1.0
    >>> 3.4//1.1
    3.0
    >>> 3.4//1.2
    2.0
    >>> -1//2
    -1
    >>> -6//2
    -3
    >>> -6//-3
    2
