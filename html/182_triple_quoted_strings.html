triple quoted string
A string which is bound by three instances of either a quotation mark (") or an apostrophe (')
While they don't provide any functionality not available with single-quoted strings, they are useful for a number of reasons.
They allow you to include unescaped single and double quotes within a string and they can span multiple lines without the use of the continuation character, making them especially useful when writing docstrings.
      
    print(""" This is a \t\t  \"\"\" quoted string where I can

    use unescaped "" (double quotes) and \n\n\n\t newlines!
    """)


    //  This is a 		  """ quoted string where I can

    //  use unescaped "" (double quotes) and 


    //         newlines!

Example 1 (Raw triple quoted string) 

    print(r""" This is a \t\t  \"\"\" quoted string where I can

    use unescaped "" (double quotes) and \n\n\n\t newlines!
    """)

    //  This is a \t\t  \"\"\" quoted string where I can

    //  use unescaped "" (double quotes) and \n\n\n\t newlines!
Example 2 (Unicode triple quoted string) 

    print(u""" This is a \t\t  \"\"\" quoted string where I can

    use unescaped "" (double quotes) and \n\n\n\t newlines!
    """)
