Falsy
Collections

  • Empty tuples ()
  • Empty dictionaries {}
  • Empty sets set()
  • Empty strings ""
  • Empty ranges range(0), range(1,1), range(2,2)
Numbers
  • Zero of any numeric type.

  • Float: 0.0
  • Complex: 0j
Constants
  • None
  • False
Others
  • Decimal(0)
  • Fraction(0, 1)
  • objects for which obj.__bool__() returns False
  • objects for which obj.__len__() returns 0
truthy
Non-empty sequences or collections (lists, tuples, strings, dictionaries, sets).
Numeric values that are not zero.
True
Testing for truthy and falsy
Use the bool() function
bool([x]) return a Boolean value, i.e. one of True or False.
x (the argument) is converted using the standard truth testing procedure.
If x is False or omitted, this returnsFalse
Otherwise it returns True.
    bool(None)
    //  False
Useful falsies

    for item in [(x,bool(x)) for x in [range(x,x) for x in range(5)]]:
        print(item)

    //  (range(0, 0), False)
    //  (range(1, 1), False)
    //  (range(2, 2), False)
    //  (range(3, 3), False)
    //  (range(4, 4), False)

    float('nan') == float('nan')
    //  False
Useful truthies

    for item in [(x, bool(x)) for x in [[''], [0], ' ', [range(0)]]]:
       print(item)

    //  ([''], True)
    //  ([0], True)
    //  (' ', True)

    bool(float('nan'))
    //  True

Notes
truthy

By default, an object is considered true unless its class defines either a __bool__() method that returns False or a __len__()

W3Schools
Almost any value is evaluated to True if it has some sort of content.
Any string is True, except empty strings.
Any number is True, except 0.
Any list, tuple, set, and dictionary are True, except empty ones.
