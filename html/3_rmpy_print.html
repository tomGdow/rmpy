print
Multiple values can be displayed as comma-separated list, separated by single space by default
alternative separator: by adding sep parameter, eg sep='**'
invisible \n at end of line //default
Syntax
[Syntax]
Examples
Example 1 
  
  price  = 3.95
  widgets = 5

  (1) print('The price of the widget is:', price, 'and', widgets, end='!!\n')
      // print('The price of the widget is:', price,'and',widgets, end='!!\n') 

  (2) print('The price of the widget is ' + str(price) + ' and ' + str(widgets))
      // The price of the widget is 3.95 and 5

  (3) print(f'The price of the widget is {price} and {widgets}')
      // The price of the widget is 3.95 and 5
