sys.intern()
To optimize memory allocation, Python does a process called interning.
For some objects (will be discussed later), Python only stores one object on heap memory and ask different variables to point to this memory address if they use those objects.
The objects that Python does interning on them are:
  1. integer numbers [-5, 256]
  2. boolean
  3. some strings.
Interning does not apply to other types of objects such as:
  1. large integers
  2. most strings
  3. floats
  4. lists
  5. dictionaries
  6. tuples.
