int(x, base=10)
Return an integer object constructed from
  1. a number x
  2. a string 'x'
Returns 0 if no arguments are given
For floating point numbers, this truncates towards zero.
If x is not a number or if base is given, then x must be a string, bytes, or bytearray instance representing an integer literal in radix base.
Optionally, the literal can be preceded by + or - (with no space in between) and surrounded by whitespace.
str(object='')
Also str(object=b'', encoding='utf-8', errors='strict')
Return a string version of object.
If object is not provided, returns the empty string.
Otherwise, the behaviour of str() depends on whether encoding or errors is given
There are MANY string methods
hex(x)
Convert an integer number to a lowercase hexadecimal string prefixed with 0x 

          hex(255)    //  '0xff'
          hex(-42)    //  '-0x2a'
Syntax 
          Syntax One
Examples
int() 

     int('10', base=2)      //     2
     int('1000', base=2)    //     8
     int('100')             //     100
     int('100', base=2)     //     4
     int(100, base=2)       //     TypeError: int() can't convert non-string with explicit base  //Note
     int(22.4)              //     22
     int(22)                //     22
     int('100.2')           //     ValueError: invalid literal for int() with base 10: '100.2'
     int(+20)               //     20
     int('+30')             //     30
     int()                  //     0             // Note
