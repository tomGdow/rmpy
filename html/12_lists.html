Lists
lists are mutable sequences
typically used to store collections of homogeneous items (where the precise degree of similarity will vary by application).
list =[1,2,3]
Lists may be constructed in several ways
  • Using a pair of square brackets to denote the empty list: []
  • Using square brackets, separating items with commas: [a], [a, b, c]
  • Using a list comprehension: [x for x in iterable]
  • Using the type constructor: list() or list(iterable)
  • list('abc') returns ['a', 'b', 'c']
  • list( (1, 2, 3) ) returns [1, 2, 3]
Lists might contain items of different types, but usually the items all have the same type.

Examples
Example 1 
  >>> squares = [1, 4, 9, 16, 25]
  >>> squares[0]
  1
  >>> squares[-3:]
  [9, 16, 25]
  >>> squares[:]  // shallow copy // !Note
  [1, 4, 9, 16, 25]
  >>> squares + [36,49,64,81]
  [1, 4, 9, 16, 25, 36, 49, 64, 81]
Example 2 

  >>> cubes = [1, 8, 27, 65, 125]
  >>> cubes[3] = 64       // lists are MUTABLE
  >>> cubes.append(216)

  >>> cubes.append(7 ** 3)
  >>> cubes
  [1, 8, 27, 64, 125, 216, 343]
Example 3 

  >>> letters = ['a', 'b', 'c', 'd', 'e', 'f', 'g']
  >>> letters[2:5]
  ['c', 'd', 'e']
  >>> letters[2:5]=['C', 'D', 'E']
  >>> letters
  ['a', 'b', 'C', 'D', 'E', 'f', 'g']
  >>> letters[2:5]=[]        // remove them
  >>> letters
  ['a', 'b', 'f', 'g']
  >>> len(letters)
  4
