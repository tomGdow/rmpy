pow(base, exp, mod=None)
 
pow(base, exp, mod=None)
    Equivalent to base**exp with 2 arguments or base**exp % mod with 3 arguments

    Some types, such as ints, are able to use a more efficient algorithm when
    invoked using the three argument form.

    pow(2,4)                //    16
    pow(2,4,5)              //    1
    pow(2,4) % 5            //    1

    2**4 % 5                //    1

    2**4**2                 //    65536
    2**16                   //    65536

    (2**4)**2 == 2**4**2    //    False
    2**(4**2) == 2**4**2    //    True
min(...)
 
    min(...)
        min(iterable, *[, default=obj, key=func]) -> value
        min(arg1, arg2, *args, *[, key=func]) -> value

        With a single iterable argument, return its smallest item. The
        default keyword-only argument specifies an object to return if
        the provided iterable is empty.
        With two or more arguments, return the smallest argument.

    min("0Aa")   //    '0'
    min("Aa")    //    'A'

    min("Za")    //    'Z'

    min("Za", key = lambda x : x.lower())  //  'a'

    min(1, *[100,-2,3], key=lambda x: pow(x,2))  //  1
round()
 

    round(number, ndigits=None)
        Round a number to a given precision in decimal digits.

        The return value is an integer if ndigits is omitted or None. 
        Otherwise the return value has the same type as the number.
        ndigits may be negative.

        round(333.33,1)       //    333.3
        round(333.33,0)       //    333.0
        round(333.33,-1)      //    330.0
        round(333.33,-2)      //    300.0
        round(333,-2)         //    300

divmod()
divmod(x, y, /) Return the tuple (x//y, x%y). Invariant: div*y + mod == x. Take two (non complex) numbers as arguments and return a pair of numbers consisting of their quotient and remainder when using integer division. For integers, the result is the same as (a // b, a % b). For floating point numbers the result is (q, a % b), where q is usually math.floor(a / b) but may be 1 less than that. divmod(22,5) // (4, 2) divmod(22.5,5) // (4.0, 2.5) divmod(22,5) // (4, 2) divmod(22.5,5) // (4.0, 2.5) divmod(23,5) // (4, 3) type(divmod(23,5)) // <class 'tuple'>
