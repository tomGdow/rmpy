leap year
Kernighan and Richie: A year is a leap year if it is divisible by 4 but not by 100, except that years divisible by 400 are leap years.
Method One
    def isleap(arg):
      if(arg%4):
        return "Not a Leap Year"
      else:
        if(arg%100):
          return "Leap Year"
        else:
          if(arg%400):
            return "Not a Leap Year"
          else:
            return "Leap Year"
