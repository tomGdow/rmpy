Dictionary
{'a':1, 'b':2}
Think of a dictionary as a set of key: value pairs
The keys must be unique (within one dictionary).
A dictionary’s keys are almost arbitrary values
Values that are not hashable may not be used as keys.
That is values containing lists, dictionaries or other mutable types  (compared by value rather than by object identity) cannot be used as keys
Dictionaries are sometimes found in other languages as associative memories or associative arrays
Unlike sequences, which are indexed by a range of numbers, dictionaries are indexed by keys, which can be any immutable type
Strings and numbers can always be keys.
Tuples can be used as keys if they contain only strings, numbers, or tuples
If a tuple contains any mutable object either directly or indirectly, it cannot be used as a key
You can’t use lists as keys
(Lists can be modified in place using index assignments, slice assignments, or methods like append() and extend())
Performing list(d) on a dictionary returns a list of all the keys used in the dictionary, in insertion order (if you want it sorted, just use sorted(d) instead)
To check whether a single key is in the dictionary, use the in keyword.
In dictionaries, later values will always override earlier ones:
Dictionary higher priority than Set
type({}) give dictionary // Note
Dictionaries can be created by several means:

  1. Use a comma-separated list of key: value pairs within braces

        {'jack': 4098, 'sjoerd': 4127}
    {4098: 'jack', 4127: 'sjoerd'}

  2. Use a dict comprehension:

        {x: x ** 2 for x in range(10)} // {2: 4, 4: 16, 6: 36}

  3. Use the type constructor:

     
    dict([('foo', 100), ('bar', 200)]) // {'foo': 100, 'bar': 200}

    dict(foo=100, bar=200) // {'foo': 100, 'bar': 200}   // No inverted commas on key!

    BUT note: 

    dict('foo'=1, 'bar'=2)   // with inverted commas on key

    !!SyntaxError: expression cannot contain assignment, perhaps you meant "=="?
Dictionary Methods
dict.get(key, default) (get method)
get(key[, default])
Return the value for key if key is in the dictionary, else default.
If default is not given, it defaults to None, so that this method never raises a KeyError.

        mydict = {1: 'one', 2: 'two'}
        mydict.get(1)                   //     'one'
        mydict.get(3, 'not there')      //     'not there'
        mydict.get(3)                   //      No output
dict.pop(key, default)
dict.pop(key[, default])
If key is in the dictionary, remove it and return its value, else return default.
If default is not given and key is not in the dictionary, a KeyError is raised.
 
    mydict = {1: 'one', 2: 'two'}
    mydict.pop(3) KeyError:           //     3
    mydict.pop(3,100)                 //     100
    mydict.pop(3,None)                 //    !! Nothing returned if key absent
dict.clear()

Remove all items from the dictionary.

del dict[key]

Remove dict[key] from dict.

Raises a KeyError if key is not in the map.

Examples
Four ways to add to dictionary (see Python Add to Dictionary )

  1. Use update() method:

     
  cp={'ireland': 'dublin', 'england': 'london'}
  cp.update({'scotland':'glascow'})
  2. Use _setitem_() Method 
     cp.__setitem__('spain', 'madrid')
  3. Use subscript notation 
    cp['france']='paris'
  4. Use the ** operator 
     new_cp = {**cp, **{'wales':'cardiff'}}
Merge dictionaries with **kwargs (arg expansion) 
    dict_one = {'a':"thomas", 'b': 'gerard'}
    dict_two = {'a':'Mick', 'c': 'pros'}
    merged = {**dict_one, **dict_two}
    merged
    {'a': 'Mick', 'b': 'gerard', 'c': 'pros'}

(see Using the * and ** unpacking operators in python )

Get all-keys, all-values

  1. list(d) gives list of keys

        mydict={'a':1, 'b':2, 'c':3}
    list(mydict) // ['a', 'b', 'c']            // keys
    mydict       // {'a': 1, 'b': 2, 'c': 3}  // no mutation
  2. mydict.values() and list() 
        mydict.values())        // dict_values([1, 2, 3])
    list(mydict.values())   // [1, 2, 3]              // values
Delete from dictionary

  1. Use del()

        mydict = {'a': 1, 'b': 2, 'c': 3}
    del(mydict['a'])
    mydict  //  {'b': 2, 'c': 3}

  2. pop(key)

    (remove key-value pair and return value)

     
    mydict.pop('c') // 3
    mydict          // {'b': 2}
Looping

    for x,y in ndic.items():
      print(f" {y}, {x}")

    // the brave, lancelot
    // the coder, josef

    for x, y in enumerate(['tic', 'tak', 'toe']):
