logical operators
xx
and
returns True only if both operands are true
 5 < 7 and 4 < 7  // True
False and False returns False !!
(as does False or False)
or
returns True when one or both of the operands are true
5 < 3 or 3 == 3 // True
or
unary operator
negates the truth of single operand
not True // False
Syntax 
          Syntax One
Examples
and 

    True    and    True   ->    True    1 < 2 and 1 < 3
    True    and    False  ->    False
    False   and    True   ->    False
    False   and    False  ->    False   1 > 2 and 1 > 3
or 

    True    or    True   ->    True   
    True    or    False  ->    True
    False   or    True   ->    True
    False   or    False  ->    False
not 

    not (False and False)    //  True
    not (False or False)     //  True
Notes
Booleans and (1 and 0)
In python, Boolean values may also be represented numerically by 1 is True and False is 0 
             True ==  1     //   True
             False == 0     //   True
