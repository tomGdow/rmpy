sorted()
Return a new sorted list from the items in iterable
Has two optional arguments which must be specified as keyword arguments

  1. key specifies a function of one argument that is used to extract a comparison key from each element in iterable (for example, key=str.lower).

    The default value is None (compare the elements directly)

  2. reverse is a boolean value. If set to True, then the list elements are sorted as if each comparison were reversed.

The built-in sorted() function is guaranteed to be stable.
A sort is stable if it guarantees not to change the relative order of elements that compare equal
A simple ascending sort is very easy: just call the sorted() function.
It returns a new sorted list
list.sort()
Python LISTS have a built-in list.sort() method that modifies the list in-place.
It modifies the list in-place (and returns None to avoid confusion).
Usually it’s less convenient than sorted() - but if you don’t need the original list, it’s slightly more efficient.
Another difference is that the list.sort() method is only defined for lists.
In contrast, the sorted() function accepts any iterable.
Both list.sort() and sorted() have a key parameter to specify a function (or other callable) to be called on each list element prior to making comparisons.

Examples
Example 1 (Lists) 

    mylist=["tomas", "tom", "thomasdow", "Thomas"]

    sorted(mylist)                 //    ['Thomas', 'thomasdow', 'tom', 'tomas']
    sorted(mylist, key=len)        //    ['tom', 'tomas', 'Thomas', 'thomasdow']
    sorted(mylist, key=str.lower)  //    ['Thomas', 'thomasdow', 'tom', 'tomas']
    sorted(mylist, key=str.lower, reverse=True)  // ['tomas', 'tom', 'thomasdow', 'Thomas']
Example 2 (Dictionary) 

    mydict={'tom': 100, 'tomdow': 40, 'harry': 100}
    sorted(mydict)                   //     ['harry', 'tom', 'tomdow']
    sorted(mydict.values())          //     [40, 100, 100]
    sorted(mydict.items(), key=len)  //     [('tom', 100), ('tomdow', 40), ('harry', 100)]
Example 3 (Reverse Name) 

    fns=sorted('tomas')
    lns=sorted('manure')
    fns    //   ['a', 'm', 'o', 's', 't']
    ms=""
    print(ms.join(fns),"".join(lns))    //  amost aemnru
Example 4 (Strings) 

    sorted('thomas')
    //    ['a', 'h', 'm', 'o', 's', 't']
Example 5 (Strings) 

    sorted("This is a test string from Andrew".split(), key=str.lower)
    sorted("This is a test string from Andrew".split(), key=len)
    //    ['a', 'Andrew', 'from', 'is', 'string', 'test', 'This']
    //    ['a', 'is', 'This', 'test', 'from', 'string', 'Andrew']
