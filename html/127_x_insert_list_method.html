lst.insert(i, x)
Insert a new item with value x in the list before position i.
Negative values are treated as being relative to the end of the list.
            l =["thomas", "mick", "peter"]

            l.insert(0, "london")    //    ['london', 'thomas', 'mick', 'peter']
            l.insert(-1, "billy")    //    ['london', 'thomas', 'mick', 'billy', 'peter']         // Note!!
            l.insert(-2, "noel")     //    ['london', 'thomas', 'mick', 'noel', 'billy', 'peter']
