xx
str.format(*args, **kwargs)
Perform a string formatting operation.
The string on which this method is called can contain literal text or replacement fields delimited by braces {}.
Each replacement field contains either the numeric index of a positional argument, or the name of a keyword argument.
Returns a copy of the string where each replacement field is replaced with the string value of the corresponding argument

    "tho{0}a{1}".format(*["m", "s"])       //     'thomas'
    "The sum of 1 +2 is {}".format(1+2)    //     'The sum of 1 +2 is 3'
Using a Single Formatter 

  (1) print("{}, a computer science  format for geeks".format("Geeks for Geeks"))
      //   Geeks for Geeks, a computer science  format for geeks

  (2) str="This article is written in {}"
      print(str.format("Python"))    //   This article is written in Python

  (3) "Hello, I am {} years old".format(18) // 'Hello, I am 18 years old'
Multiple formatters

  (1) my_string="This is {} {} {} {}"
      print(my_string.format("one", "two", "three", "four"))
      //    This is one two three four

  (2) print("Every {} should know the use of {} {} programming and\
      {}".format("programmer", "Open", "Source", "Operating Systems"))
      // Every programmer should know the use of Open Source \
      programming and Operating Systems

  (3) print("Every {3} should know the use of {2} {1} programming and\
      {0}".format("programmer", "Open", "Source", "Operating Systems"))
      // Every Operating Systems should know the use of Source Open\
      programming and programmer

  (4) my_string_two="{gfg} is a {0} science portal for {1}"
      my_string_two.format("computer", "geeks", gfg="GeeksforGeeks")
      // 'GeeksforGeeks is a computer science portal for geeks'
Type Specifying
More parameters can be included within the curly braces.
Use the format code syntax {field_name:conversion}
Where field_name specifies the index number of the argument to the str.format() method
And conversion refers to the conversion code of the data type.
  (1) print("The temperature today is {0:d} degrees outside!".format(35.567))
      //  ValueError: Unknown format code 'd' for object of type 'float'

      print("The temperature today is {0:f} degrees outside!".format(35.567))
      // The temperature today is 35.567000 degrees outside!

      print("The temperature today is {0:.2f} degrees outside!".format(35.567))
      // The temperature today is 35.57 degrees outside!

  (2) print("My average for this {0} was {1:.0f}%".format("semester",78.234876))
      //  My average for this semester was 78%

  (3) print("The {0} form of 8 is {1:b}".format("binary", 8))
      // The binary form of 8 is 1000
Padding Substitutions or Generating Spaces
By default, strings are left-justified within the field,
By default, numbers are right-justified.
We can modify this by placing an alignment code just following the colon.
< left-align text in the field
^ center text in the field
> right-align text in the field
'=' Forces the padding to be placed after the sign (if any) but before the digits

  (1) print("{0:^16} was founded in {1:<16}!".format("GeeksforGeeks", 2009))
      print("{0:<16} was founded in {1:>16}!".format("GeeksforGeeks", 2009))
      print("{0:>16} was founded in {1:^16}!".format("GeeksforGeeks", 2009))

       GeeksforGeeks   was founded in 2009            !
      GeeksforGeeks    was founded in             2009!
         GeeksforGeeks was founded in       2009      !

  (2) print("{:*^20s}".format("Geeks"))
        // *******Geeks********
More good examples
  (1) print("{:+f}, {:+f}".format(3.14, -3.14))  //  +3.140000, -3.140000

  (2) print("{:,}".format(123456789))   //  123,456,789
                                      //  use comma as thousands separator
