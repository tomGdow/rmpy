bytes literal
A prefix of 'b' or 'B' indicates that the literal should become a bytes literal in Python 3
bytes = b'...' literals = a sequence of octets (integers between 0 and 255)
How to define byte "strings"
            // David Beazley, SlideShare ref

            a = b"ACME 50 91.10"  // Byte string literal
                                  // b'ACME 50 90.10'

            b= bytes([1,2,3,4,5]) // From a list of integers
                                  // b'\x01\x02\x03\x04\x05'

            c = bytes(10)         // An array of 10 zero-bytes
                                  // b'\x00\x00\x00\x00\x00\x00\x00\x00\x00\x00'
            
            d=bytes('Jalepño', 'utf8') // Encoded from string
                                       // b'Jalep\xc3\xb1o'

            bytes.fromhex('c2a9ff')  // From a string of hex digits 
                                     // b'\xa2\xc8\xff'

            // use ascii() to see escapes

            ascii('Jalepño')         // "'Jalep\\xf1o'"

Examples
Example 1 

    >>> myby = b"0Aa"
    >>> for x in myby:
    print(x)

    48
    65
    97
Example 2 

    >>> myby_alt=bytes('0Aa','ascii')
    >>> myby_alt
    b'0Aa'
    >>> myby_alt is myby
    False
    >>> myby == myby_alt
    True
    >>> for x in myby_alt:
    print(x)

    48
    65
    97
