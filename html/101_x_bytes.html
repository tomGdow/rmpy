bytes literal
A prefix of 'b' or 'B' indicates that the literal should become a bytes literal in Python 3
bytes = b'...' literals = a sequence of octets (integers between 0 and 255)
Syntax 
          Syntax One
Examples
Example 1 

    >>> myby = b"0Aa"
    >>> for x in myby:
    print(x)

    48
    65
    97
Example 2 

    >>> myby_alt=bytes('0Aa','ascii')
    >>> myby_alt
    b'0Aa'
    >>> myby_alt is myby
    False
    >>> myby == myby_alt
    True
    >>> for x in myby_alt:
    print(x)

    48
    65
    97
