is / is_not operators
The operators is and is not test for object's identity:
x is y is true if and only if x and y are the same object.
An Object’s identity is determined using the id() function.
So, A is B is the same as id(A) == id(B)
x is not y yields the inverse truth value.

Examples
Example 1 

  >>> a =[10,20,30]
  >>> b = [10,20,30]
  >>> a is b
  False
  >>> a == b
  True
  >>> c = a
  >>> c
  [10, 20, 30]
  >>> c is a
  True
  >>> id(a)
  1716773964992
  >>> id(b)
  1716747401024
  >>> id(c)
  1716773964992
Example 2 

  >>> x = 'a' 
  >>> x += 'bc'
  >>> y = 'abc'
  >>> x == y // True
  >>> x is y // False

  >>> z = 'abc'
  >>> w = 'abc'
  >>> z is w // True
