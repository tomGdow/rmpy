None
None is a special constant in Python that represents the absence of a value or a null value.
It is an object of its own datatype, the NoneType.
We cannot create multiple None objects but can assign it to variables.
These variables will be equal to one another.
We must take special care that None does not imply False, 0 or any empty list, dictionary, string etc.
The Python None object, denoting lack of value.
This object has no methods.
It needs to be treated just like any other object with respect to reference counts.
