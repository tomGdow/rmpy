zip
zip(*iterables)
Make an iterator that aggregates elements from each of the iterables.
The iterator stops when the shortest input iterable is exhausted
With a single iterable argument, it returns an iterator of 1-tuples.
With no arguments, it returns an empty iterator
zip() in conjunction with the * operator can be used to unzip a list:
If only one iterable object passed to the zip() function then get a list of 1-item tuples
Succinct summary
The zip function will return an iterator of tuples, where the i-th tuple will contain the i-th element from each of the iterables passed in.
This iterator will stop once the shortest input iterable has been exhausted.

Examples
Example 1 
    a =("Thomas", "Josef", "Emer")
    b = ("D", "Z", "G","H")
    zed = zip(a,b)
    zed
    <zip at 0x149973d2580>
    tuple(zed)
    (('Thomas', 'D'), ('Josef', 'Z'), ('Emer', 'G'))
Example 2 (Unzip) 
            a =("Thomas", "Josef", "Emer")
            b = ("D", "Z", "G","H")
            zed = zip(a,b)
            tzed=tuple(zed)
            tzed
            // (('Thomas', 'D'), ('Josef', 'Z'), ('Emer', 'G'))
            aa, bb = zip(*tzed)
            aa
            // ('Thomas', 'Josef', 'Emer')
            bb
            // ('D', 'Z', 'G')
            zed
            // <zip at 0x14999d88d00>
            tuple(zed)
            // ()
Example 3 (For Loop) 
    fnames=["tom","josef", "emer"]
    lnames="dow", "zac", "mor"
    age=12, 14, 100

    for f, l, a in zip(fnames, lnames, age):
      print(f"{l}, {f} is age {a}")


    //  dow, tom is age 12
    //  zac, josef is age 14
    //  mor, emer is age 100
Example 4 (Single argument) 

    for f in zip(fnames):
      print(f"{f}")


    //  ('tom',)
    //  ('josef',)
    //  ('emer',)

    for f, in zip(fnames):
      print(f"{f}")

    //  tom
    //  josef
    //  emer
