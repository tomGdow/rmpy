memoryview
A memoryview object exposes the C level buffer interface as a Python object which can then be passed around like any other object.
memoryview objects allow Python code to access the internal data of an object that supports the buffer protocol without copying.
memoryview(obj): Create a memoryview that references obj.
obj must support the buffer protocol.
Built-in objects that support the buffer protocol include bytes and bytearray.
A memory view is a safe way to expose the buffer protocol in Python.
It allows you to access the internal buffers of an object by creating a memory view object.

Examples
Example 1 

    myba=bytearray("0Aa", 'utf-8')
    myba    //    bytearray(b'0Aa')

    mv=memoryview(myba)
    mv      //    <memory at 0x00000276369E1100>
    mv[1]   //    65
    mv[1]=66

    myba    // bytearray(b'0Ba')    // Mutated
