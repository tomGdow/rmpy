any (built-in)
xx
any(iterable)
Return True if any element of the iterable is true.
If the iterable is empty, return False.
Equivalent to:

  def any(iterable):
    for element in iterable:
      if element:
        return True
    return False
Syntax 
          Syntax One
Examples
Example 1 (my_any code challenge) 

  def my_any (iter):

      ''' Return True if any element of the iterable is true. If the iterable is
          empty, return False. '''

      for i in iter:
        if i:
          return True;
        else:
          continue;
      return False
