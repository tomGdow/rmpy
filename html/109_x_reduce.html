reduce
reduce(function, iterable)
The reduce function applies two arguments cumulatively to the items of iterable, from left to right
Syntax 
          Syntax One
Examples
Example 1 
    lst = [10,6,7,5,2,1,8,5]
    reduce(lambda x,y : x if (x>y) else y, lst)  //  10
Example 2 
    lst = [10,6,7,5,2,1,8,5]
    reduce(lambda x,y: x+y, lst) // 44
