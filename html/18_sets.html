Sets
A set object is an unordered collection of distinct hashable objects.
Common uses include
  1. membership testing
  2. removing duplicates from a sequence
  3. computing mathematical operations such as intersection, union, difference
sets support
  1. x in set
  2. len(set)
  3. for x in set.
sets do not record element position or order of insertion.
sets do not support indexing, slicing, or other sequence-like behavior.
There are currently two built-in set types:
set
  1. The set type is mutable
  2. set contents can be changed using methods like add() and remove()
  3. set has no hash value and cannot be used as either a dictionary key or as an element of another set.
frozenzet
  1. The frozenset type is immutable and hashable
  2. frozenset contents cannot be altered after it is created;
  3. frozenset can therefore be used as a dictionary key or as an element of another set
Sets may be created by several means:
  1. Use a comma-separated list of elements within braces: {'jack', 'sjoerd', 'jack'}
  2. Use a set comprehension: {c for c in 'abracadabra' if c not in 'abc'}
Use the type constructor:
  1. set()
  2. set('foobar')  // {'b', 'f', 'r', 'a', 'o'}
  3. set(['a', 'b', 'foo'])
dictionary has higher priority than set
Note: to create an empty set you have to use set(), not {} ; the latter creates an empty dictionary
Set Methods
myset.add(elem) (Add to Set) 
 
    # add elem to set

    myset={"tom", "dick", "harry"}    //  
    myset.add("dirty")                //    {'tom', 'dick', 'dirty', 'harry'}
myset.remove(elem) (Remove from Set) 

    # remove elem from set

    myset =  {'tom', 'dick', 'dirty', 'harry'} 
    myset.remove("dirty")                //    {'tom', 'dick', 'harry'}

    # note  (and compare with myset.discard())  

    myset.remove('annie')    //  KeyError: 'annie'
myset.discard(elem) (Discard from Set) 

    # discard elem from set

    myset =  {'tom', 'dick', 'dirty', 'harry'} 

    # note  (and compare with myset.remove())  

    myset.discard('dirty')    //     {'tom', 'dick', 'harry'}
    myset.discard('annie')    //     NO ERROR MESSAGE
    myset                     //     {'tom', 'dick', 'harry'}
myset.update() (Update Set) 

            # update a set content

             myset={"tom", "dick", "harry"}    //  {'tom', 'dick', 'harry'}
             myset.update(['jones', 'tupin', 'worth'])
                  //    {'dick', 'worth', 'jones', 'harry', 'tom', 'tupin'}
            
            # Example 2

            myset={"Tom", "Dick", "Harry"}
            myset    //   {'Harry', 'Tom', 'Dick'}
            myset.update(['Worth'])
            myset   //    {'Worth', 'Harry', 'Tom', 'Dick'}
            myset.update('hello')
            myset   //    {'h', 'Tom', 'Dick', 'e', 'l', 'o', 'Worth', 'Harry'}   //  !Note

            # Example 3 

            myset.clear()
            myset.add('thomas')
            myset                      // {'thomas'}
            myset.update('thoma)
            myset                      // {'h', 'a', 'thomas', 'm', 't', 'o', 's'}
                                       //  ! Compare with above
set.union() or | 
    s2={4,5,6,7,8}

    s1.union(s2)    //  {1, 2, 3, 4, 5, 6, 7, 8}
    s1 | s2         //  {1, 2, 3, 4, 5, 6, 7, 8}
set.intersection() or & 

    s1={1, 2, 3, 4, 5}                                                
    s2={4,5,6,7,8}

    s1.intersection(s2)        //     {4, 5}
    s1&s2              //     {4, 5}
set.difference() or - 

//  Returns a set containing elements only in the first set, but not in the second set

    s1={1, 2, 3, 4, 5}                                                
    s2={4,5,6,7,8}

    s1.difference(s2)       //  {1, 2, 3}
    s1-s2                   //  {1, 2, 3}
set.symmetric_difference() or ^ 

    Returns a set consisting of elements in both sets, excluding the common elements.

    s1={1, 2, 3, 4, 5}                                                
    s2={4,5,6,7,8}

    s1^s2                        //    {1, 2, 3, 6, 7, 8}
    s1.symmetric_difference(s2)  //    {1, 2, 3, 6, 7, 8}

Example 2 
    basket = {'apple', 'orange', 'apple', 'pear', 'orange', 'banana'}
        {'banana', 'orange', 'pear', 'apple'}   // No duplicates
    'orange' in basket
        True
    'crabgrass' in basket
        False

    a = set('abracadabra')
    a  // {'c', 'a', 'r', 'd', 'b'}  // unique letters in a
    b = set('alacazam')             
    b // {'c', 'l', 'z', 'm', 'a'}   
    a-b // {'d', 'b', 'r'}            // letters in a but not in b  
    a|b // {'c', 'l', 'z', 'a', 'm', 'r', 'd', 'b'}  //  letters in a or b or both     
    a&b // {'c', 'a'}                        //  letters in both a and b
    a^b // {'r', 'd', 'b', 'l', 'z', 'm'}   // letters in a or b but not both

Notes
Equality of Sets
Both set and frozenset support set-to-set comparisons.
Two sets are equal if and only if every element of each set is contained in the other (each is a subset of the other).
A set is less than another set if and only if the first set is a proper subset of the second set (is a subset, but is not equal).
A set is greater than another set if and only if the first set is a proper superset of the second set (is a superset, but is not equal).
