bool()
xx
bool(x)
Return a Boolean value, i.e. one of True or False.
x is converted using the standard truth testing procedure.
If x is false or omitted, this returns False; otherwise it returns True.
Syntax 
          Syntax One
Examples
Example 1 
      bool('')  //  False
      bool(' ')  //  True
